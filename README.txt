
# Predador de Padrões - por Rogger

Este é um robô automatizado para operar na plataforma Deriv usando análise de comportamento humano (estratégia anti-manada).

## Como usar:

1. Gere seu token (conta demo ou real) no site da Deriv.
2. Suba esse projeto no Streamlit Cloud ou rode localmente.
3. Insira o token na interface, configure os parâmetros e clique em "Iniciar Robô".

Configurações padrão:
- Stake: R$1,00
- Martingale: 2.0
- Stop Loss: R$60,00
- Meta de lucro: R$50,00
- Delay entre entradas: 7 segundos
